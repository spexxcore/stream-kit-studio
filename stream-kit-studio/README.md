# Stream Kit Studio

AI-powered stream branding generator. Describe your brand, get a full kit: logo, overlays, alerts, panels, and banner — all generated by Claude + Flux AI.

## Stack
- React + Vite → Netlify
- Netlify Functions (serverless)
- Anthropic Claude (creative direction + HTML/CSS generation)
- fal.ai Flux (logo + mascot image generation)

## Setup

### 1. Clone and install
```bash
npm install
```

### 2. Add environment variables
In Netlify Dashboard → Site Settings → Environment Variables:
```
ANTHROPIC_API_KEY=sk-ant-...
FAL_API_KEY=...
```

For local dev, create a `.env` file (Netlify CLI reads it):
```
ANTHROPIC_API_KEY=sk-ant-...
FAL_API_KEY=...
```

### 3. Local dev
```bash
npm install -g netlify-cli
netlify dev
```
Runs at `http://localhost:8888`

### 4. Deploy
Push to GitHub, connect to Netlify, it auto-deploys.

## How it works

1. User fills out 4-step intake form (brand name, mascot, vibe, style)
2. `generate-kit` function → Claude builds brand brief + image prompts → fal.ai generates logo + mascot images
3. Kit page shows brand colors, fonts, generated images
4. User clicks any asset type (overlay, alerts, panels, banner)
5. `generate-overlay` function → Claude writes full HTML/CSS file for that asset
6. User previews in iframe or views source code, then downloads the HTML file

## Getting a fal.ai API key
1. Go to [fal.ai](https://fal.ai)
2. Create account → API Keys → Create key
3. Free tier includes enough credits to test

## Asset outputs
- **Logo/Mascot**: PNG images from Flux Schnell (fast, high quality)
- **Overlay**: Single HTML file for OBS browser source (1920×1080, transparent bg)
- **Alerts**: HTML file with Follow/Sub/Donation alert previews + CSS animations
- **Panels**: HTML file with 6 styled channel info panels
- **Banner**: HTML file with 1200×480 channel banner + download button
